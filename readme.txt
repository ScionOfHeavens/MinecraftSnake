работа была проведена в visual studio code
ЗАПУСК производится в builder.py

4 блока появившиеся у персонажа: золото, алмаз, камень, железо 
                - управление вверх, вниз, влево, вправо

tnt блоки - яблоки, которые надо собирать

столкновение со своим хвостом или границей приводит к завершению игры и печатанью финального счета

змея растетпри поедании tnt

команда в чат "Finish game" завершает игру досрочно

snake - змея, fence - граница, game - содержит главный цикл, apple - яблоко
input системы работают в отдельных классах
collide system проверяет коллизии объектов, и вызвает взаимодействие между объектами с коллизией
my_mcpi_and_stuff - некоторые нароботки, не все используется
static_objects - ещё одна папка почти утратившая свою идею и ценность, там находится инициализация соединения с minecraft
